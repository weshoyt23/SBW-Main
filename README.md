# SBW Co. — Command Center

A collection of self-contained HTML pages that make up the SBW Co. command center.
Each file is a standalone page — open it directly in a browser, no build step or
server required.

## Getting started

Open [`index.html`](index.html) in your browser to reach the command center
landing page, which links out to the rest of the workspace.
